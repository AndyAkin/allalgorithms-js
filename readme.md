## Install

```
npm install allalgorithms
```

## Usage Example

```js
const allalgorithms = require('allalgorithms');

arr = [77, 2, 10, -2, 1, 7]

console.log(allalgorithms.sorting.bubbleSort(arr));
// -> [-2, 1, 2, 7, 10, 77]
```
